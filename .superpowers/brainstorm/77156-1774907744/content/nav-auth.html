A — Ocultar seções de membro

Sem login: só aparece conteúdo público (Bíblia, Hinário, Manual). Um botão CTA "Entrar na minha igreja" no rodapé convida ao login.

Prós

  • Interface limpa
  • Sem confusão

Contras

  • Usuário não sabe o que existe na área de membro

B — Mostrar itens bloqueados

Itens da área de membro ficam visíveis mas desabilitados/acinzentados com ícone de cadeado. Clicar redireciona para login.

Prós

  • Usuário descobre o que existe
  • Incentiva o cadastro

Contras

  • Interface mais carregada

C — Entrada única "Minha Igreja"

Um único item especial no rodapé da sidebar agrupa toda a área de membro. Após login, expande para mostrar todos os itens (Oração, Eventos, Diretório, etc.).

Prós

  • Interface mais limpa
  • Convite claro ao login
  • Revela conteúdo progressivamente

Contras

  • Um clique a mais para navegar

Clique em uma opção para selecionar • Isso se aplica tanto à sidebar (desktop) quanto ao bottom nav (mobile)
